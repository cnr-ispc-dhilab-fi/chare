# Sense of Care for Cultural Heritage

This repository contains the material produced in the research on sense of care on Cultural Heritage (CH). 

### CHARE Scale

The repository hosts the supplementary material for the  Scale for the Assessment of Caring for Cultural Heritage (CHARE)

Read full validation study on MDPI: Veggi, M.; von Gal, A.; Piccardi, L.; Pescarin, S.; Nori, R. How Much Do We Care About Cultural Heritage? A Rasch Scale Validation Study Among Young Adults. *Heritage* 2025, 8, 317.
